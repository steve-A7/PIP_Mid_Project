--Instructions for running Inventory & Billing System Application--

I. Create venv environment:
py -m venv .venv
or
python3 -m venv .venv

II. Activate venv environment:

# Windows

.venv\Scripts\activate

# MacOS/Linux

source .venv/bin/activate

III. Install dependencies:
python -m pip install -r requirements.txt

IV. Install Custom Fonts:
Goto Assets Folder and install "Caveat-Regular.ttf", "Manrope-ExtraBold.ttf", "Poppins-Regular.ttf"

V. Run the application:
In the .venv environment run the main file with the command:
python main.py
